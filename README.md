# systems-analysis-dashboard 
# Исследовательский проект: Интерактивный дашборд "Игра Престолов"

Данный репозиторий содержит материалы лабораторных работ №10 и №11 по анализу данных и проектированию UI/UX интерфейсов.

## 📊 О проекте
Цель работы — разработка интерактивного дашборда для визуализации и анализа сквозной статистики, оценок и просмотров эпизодов вселенной "Game of Thrones" в реальном времени с использованием каскадной фильтрации.

## 🛠️ Стек технологий
* **Python 3.12** (Анализ и обработка данных)
* **Pandas** (Очистка, приведение типов и вычисление Popularity Score)
* **Streamlit** (Построение реактивного веб-интерфейса и кэширование)
* **Plotly Express** (Интерактивная визуализация: линейные графики, столбчатые диаграммы)

## 📈 Реализованный функционал
1. **Каскадные фильтры:** Главный фильтр по Сезонам, динамически пересчитывающий доступных Режиссеров на боковой панели.
2. **KPI-карточки:** Мониторинг общего числа серий, среднего рейтинга IMDb, пика просмотров в США и среднего хронометража.
3. **Визуализация трендов:** Интерактивный линейный график динамики аудитории и столбчатая диаграмма распределения оценок.
4. **Интерактивная таблица:** Вывод ТОП-10 эпизодов на основе кастомного вычисляемого индекса популярности.

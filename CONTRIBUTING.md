# Правила участия в проекте (Contributing Guidelines)

Приветствуем вас в исследовательском проекте дашборда "Игра Престолов"! Чтобы совместная работа была эффективной, придерживайтесь следующих правил:

## 🌿 Работа с ветками (Branching)
* Запрещено делать прямые коммиты в ветку `main`.
* Любая новая задача (feature, bugfix, documentation) разрабатывается в отдельной ветке.
* Шаблон названия веток: `feature/имя-задачи` или `research/тема-анализа`.

## 🔄 Направление изменений (Pull Requests)
1. Перед отправкой кода убедитесь, что дашборд Streamlit запускается локально без ошибок.
2. Создайте Pull Request (PR) из своей рабочей ветки в ветку `main`.
3. Назначьте хотя бы одного участника для проведения Code Review.
4. Слияние (merge) разрешено только после успешного одобрения ревьюером.

## 📝 Требования к коммитам
* Пишите информативные сообщения к коммитам на русском или английском языке (например: `feat: добавлен график распределения оценок`, `docs: обновлена инструкция по запуску`).
